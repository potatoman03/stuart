## Provenance rules

- Every factual claim in every saved file must cite a specific source.
- If sources disagree, surface the disagreement explicitly.
- If you infer across multiple sources, mark it as synthesis and cite every contributing source.
- Do not claim to have read a source you did not fetch.
- Do not upgrade hedged claims into stronger ones.

### Required uncertainty markers

- `> **Sources disagree:** ...`
- `> **Uncertain:** ...`
- `> **Author's synthesis:** ...`
