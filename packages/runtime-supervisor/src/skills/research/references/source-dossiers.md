## Source dossier format

Each saved source dossier should be a substantial teaching document, not a link dump.

Required structure:

1. YAML frontmatter with `title`, `sources`, and `fetched`
2. Short prerequisites section
3. Main concepts with claim-level citations
4. At least 2 hands-on exercises when the topic supports it
5. Key takeaways

Keep each file self-contained and substantial. Prefer fewer high-quality dossiers over many thin ones.
